# WeeklyPulse Setup Guide

## Backend
- cd backend
- npm install
- npm run dev

## Frontend
- cd frontend
- npm install
- npm run dev

## Notes
- The backend is using an initial Express structure with routes, controllers, services, middleware, validations, and utilities.
- The frontend is using a Vite + React + Tailwind structure with components, pages, services, context, hooks, and routes.
